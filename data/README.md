# Datasets

Place your datasets here:

- `UCI/` - UCI social network dataset
- `Hepth/` - Hepth citation network dataset  
- `Wikipedia/` - Wikipedia co-editing network dataset
- `Reddit/` - Reddit hyperlink network dataset

Each dataset should contain:
- `edges.txt` - Edge list (source, target, timestamp)
- `nodes.txt` - Node list

Format:
```
edges.txt:
node1 node2 timestamp
...

nodes.txt:
node_id
...
```
